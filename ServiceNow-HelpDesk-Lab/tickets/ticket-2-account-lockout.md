# Ticket 2: Account Lockout

**Incident ID:** INC0010002

## Issue
User account locked after multiple failed login attempts.

## User Report
User states they attempted login multiple times and are now locked out.

## Troubleshooting Steps
1. Checked account status in Active Directory
2. Confirmed account lockout
3. Unlocked account
4. Verified successful login

## Resolution
Account unlocked successfully.

## Root Cause
Multiple incorrect password attempts triggered security lockout policy.
