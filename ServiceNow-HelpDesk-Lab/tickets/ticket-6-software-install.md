# Ticket 6: Software Installation

**Incident ID:** INC0010006

## Issue
User requesting Adobe Reader installation.

## User Report
User unable to open PDF files.

## Troubleshooting Steps
1. Verified user permissions
2. Downloaded latest Adobe Reader installer
3. Installed software
4. Verified PDF file opens correctly

## Resolution
Adobe Reader installed successfully.

## Root Cause
Required software not installed on workstation.
