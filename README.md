
# ServiceNow Help Desk Lab

This project documents my hands-on ServiceNow practice using a personal developer instance. I created and managed incident tickets to simulate common IT support tasks and improve my understanding of enterprise help desk workflows.

## Skills Demonstrated

- Incident ticket creation
- Troubleshooting documentation
- Ticket lifecycle management
- Root cause analysis
- IT support workflow simulation

## Ticket Scenarios

- Password reset
- Account lockout
- Outlook not opening
- Printer offline
- Network drive not accessible
- Software installation request
