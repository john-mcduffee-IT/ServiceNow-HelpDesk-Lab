# Ticket 4: Printer Offline

**Incident ID:** INC0010004

## Issue
Department printer showing offline.

## User Report
Multiple users unable to print to shared printer.

## Troubleshooting Steps
1. Verified printer network connectivity
2. Restarted print spooler service
3. Restarted printer device
4. Tested print job

## Resolution
Printer returned to online status and printing successfully.

## Root Cause
Print spooler service stopped on workstation.
