# Ticket 1: Password Reset

**Incident ID:** INC0010001

## Issue
User unable to log into workstation after password expiration.

## User Report
User reports that they cannot log into their workstation and receive an authentication error.

## Troubleshooting Steps
1. Verified account status in Active Directory
2. Confirmed password expiration
3. Reset password and required password change at next login

## Resolution
Password reset successfully in Active Directory.

## Root Cause
Password expiration policy triggered account authentication failure.
